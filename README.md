# ChatGPT Clone

This project is a ChatGPT clone powered by Google's Gemini API, providing a simple and interactive user interface. The project was built using a template from [Assistant UI](https://assistant-ui.com) and deployed using Vercel.

## Features

- Interactive ChatGPT-like interface.
- Integrated with the **Google Gemini API** (using the free tier).
- Responsive design for an optimal user experience.

## Technologies Used

- **Frontend:** Custom UI based on the Assistant UI template.
- **API:** Google Gemini API for chat functionality.
- **Deployment:** Vercel for easy deployment and hosting.
- **Development Environment:** Visual Studio Code.

## Installation

To run this project locally, follow these steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/chatgpt-clone.git
    ```

2. Navigate to the project directory:
    ```bash
    cd chatgpt-clone
    ```

3. Install the necessary dependencies:
    ```bash
    npm install
    ```

4. Start the project locally:
    ```bash
    npm start
    ```

Make sure you have **Node.js** and **npm** installed on your machine.
